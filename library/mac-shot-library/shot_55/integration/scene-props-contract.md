# Scene Props Contract

Shot 55 can be adapted into a configurable AI input-to-generated-workspace scene.

## Required

- AI label or feature tag.
- Prompt/input text.
- Generation status text.
- Generated workspace rows and summary lines.
- 16:9 Remotion composition with safe top and side margins.

## Optional

- Feature title.
- Prompt typing speed.
- Workspace/sidebar labels.
- Row labels.
- Status chip copy.
- Accent colors.

Preview media and extracted reference clips are intentionally not included in this package.
