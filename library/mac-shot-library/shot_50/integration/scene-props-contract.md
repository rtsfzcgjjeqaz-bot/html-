# Scene Props Contract

Shot 50 can be adapted into a configurable app-grid and dashboard reveal scene.

## Required

- Headline text.
- Main dashboard shell with nav labels, heatmap cells, KPI metrics, and insight widgets.
- 16:9 Remotion composition with safe top and side margins.

## Optional

- Product/browser title.
- Sidebar labels.
- KPI labels and values.
- Heatmap intensity matrix.
- Gauge value.
- Module card titles and accent colors.
- Camera push intensity.

Preview media and extracted reference clips are intentionally not included in this package.
