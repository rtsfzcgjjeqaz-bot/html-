# Scene Props Contract

Current implementation renders with built-in safe defaults.

Optional future props for Windows adaptation:

- headline: feature-stage headline.
- subheadline: small URL or supporting line.
- mobileTitle: mobile UI title.
- desktopTitle: desktop panel title.
- featureCards: desktop feature card labels or states.
- accentGradient: brand highlight gradient.

Production integration should validate title length, dark-mode contrast, and dual-device safe area.
