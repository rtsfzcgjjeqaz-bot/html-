# Atomic Motions

- document-surface-tilt-in: dark document editor enters as the work surface.
- paragraph-rows-resolve: document paragraph rows resolve with structure.
- selection-highlight-sweep: selected text highlight sweeps across target rows.
- ai-toolbar-dock-reveal: vertical AI toolbar docks beside the selection.
- command-tooltip-pop: compact AI command tooltip appears.
- selection-focus-settle: selected recommendation state holds.
