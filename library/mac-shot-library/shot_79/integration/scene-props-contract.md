# Scene Props Contract

Shot 79 can be adapted into a configurable AI suggestion card generated from selected text.

## Required

- Document context rows.
- Selected text range.
- Toolbar action labels.
- Suggestion card title.
- Suggestion copy lines.

## Optional

- Accent glow color.
- Toolbar icon set.
- Suggestion card position.
- Tooltip labels.
- Card expansion timing.

Preview media and extracted reference clips are intentionally not included in this package.
