# Atomic Motions

- selected-document-context: establishes selected document text as the semantic anchor.
- toolbar-action-dock: docks the AI toolbar beside the selected text.
- edit-command-tooltip-cycle: reveals compact edit command labels.
- suggestion-card-expand: expands the AI suggestion result card.
- suggestion-copy-resolve: resolves suggested copy inside the card.
- suggestion-focus-settle: holds the completed suggestion state.
