# Scene Props Contract

Current implementation renders with built-in safe defaults.

Optional future props for Windows adaptation:

- promptText: short user query or command.
- workspaceTitle: email or document surface title.
- draftTitle: generated result heading.
- rows: generated content row widths or copy descriptors.
- assistantLabel: right rail label.
- accentColor: semantic highlight color.

Long text should be constrained, truncated, or measured before runtime integration.
