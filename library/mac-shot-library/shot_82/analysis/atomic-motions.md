# Atomic Motions

- editor-toolbar-context-resolve: resolves the dark editor context and toolbar.
- segmented-toggle-arrive: brings the Code / Preview segmented control into focus.
- cursor-preview-target: moves the cursor toward the Preview segment.
- preview-toggle-activate: activates Preview with a checkmark and filled segment.
- focus-lens-expand: expands a circular focus lens around the active toggle.
- preview-state-settle: holds the activated Preview state.
