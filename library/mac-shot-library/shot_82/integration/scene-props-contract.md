# Scene Props Contract

Shot 82 can be adapted into a configurable code-to-preview mode switch micro-interaction.

## Required

- Left segment label.
- Right segment label.
- Active segment.
- Cursor target point.
- Toolbar icon set.
- Background code rows.

## Optional

- Segment accent colors.
- Lens size and position.
- Cursor symbol.
- Code row density.
- Editor crop and tilt.

Preview media and extracted reference clips are intentionally not included in this package.
