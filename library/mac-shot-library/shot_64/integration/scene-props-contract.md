# Scene Props Contract

Shot 64 can be adapted into a configurable dark result-comparison impact card.

## Required

- Active card label.
- Active card number or sequence marker.
- Supporting background card labels and numbers.
- 16:9 Remotion composition with safe card bounds.

## Optional

- Card imagery or product-safe visual placeholders.
- Background card count.
- Accent glow color.
- Active card scale.
- Hold duration.

Preview media and extracted reference clips are intentionally not included in this package.
