# Scene Props Contract

Current implementation renders with built-in safe defaults.

Optional future props for Windows adaptation:

- coreLabel: optional central module label.
- nodeLabels: optional labels for surrounding workflow nodes.
- accentGradient: central module gradient.
- railCount: constrained connection rail count.
- backgroundTone: dark stage color token.

Production integration should keep rails/nodes semantic and avoid decorative overpopulation.
