# Scene Props Contract

Shot 51 can be adapted into a configurable metric-insight dashboard scene.

## Required

- Main dashboard shell.
- Hero object or product placeholder.
- KPI metric values and labels.
- Timeline/Gantt bars with labels.
- 16:9 Remotion composition with safe top and side margins.

## Optional

- Headline text.
- Warm or neutral background tone.
- Metric colors.
- Timeline group labels.
- Supporting analysis rows.
- Page-stack/fanout intensity.

Preview media and extracted reference clips are intentionally not included in this package.
