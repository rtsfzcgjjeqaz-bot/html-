# Scene Props Contract

Current implementation renders with built-in safe defaults.

Optional future props for Windows adaptation:

- promptText: user prompt copy.
- placeholderText: composer placeholder.
- attachedMediaLabel: semantic media descriptor.
- toolLabels: composer tool labels.
- sendLabel: submit affordance label.
- accentColor: send/highlight color.

Prompt text must be measured or clamped before production runtime integration.
