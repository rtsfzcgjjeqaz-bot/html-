# Scene Props Contract

Shot 81 can be adapted into a configurable prompt-to-code or AI coding assistant demo.

## Required

- Short headline text.
- Highlight keyword.
- Code editor title or file name.
- Code row model with indentation, widths, and semantic colors.
- Selection block target row.

## Optional

- Background glow color.
- Editor tilt and scale.
- Code row reveal stagger.
- Selection highlight color.
- Final settle strength.

Preview media and extracted reference clips are intentionally not included in this package.
